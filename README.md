# Simple-CLI-Calculator
A robust command-line interface (CLI) calculator built with Python. This tool supports basic arithmetic, power functions, remainders, and maintains a session history of all your calculations.

Basic Arithmetic: Addition, Subtraction, Multiplication, and Division.

Advanced Operations: Exponentiation (Power) and Modulus (Remainder).

Session History: View a list of all calculations performed during the current session.

Input Validation: Handles non-numeric inputs and prevents crashes on division by zero.

Flow Control: Terminate or reset the calculator at any time using special characters.
